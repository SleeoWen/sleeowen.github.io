keep-alive

  • vueеҶ…зҪ®з»„д»¶пјҢиғҪеңЁз»„д»¶еҲҮжҚўиҝҮзЁӢдёӯе°ҶзҠ¶жҖҒдҝқз•ҷеңЁеҶ…еӯҳдёӯпјҢйҳІжӯўйҮҚеӨҚжёІжҹ“DOM
  • keep-aliveжҳҜз”ЁеңЁдёҖдёӘзӣҙеұһеӯҗз»„д»¶иў«ејҖе…ізҡ„жғ…еҪўпјҢеҗҢж—¶еҸӘжңүдёҖдёӘеӯҗз»„д»¶еңЁжёІжҹ“пјҢиӢҘжңүv-forеҲҷдёҚдјҡе·ҘдҪң

жіЁпјҡkeep-aliveжҳҜдёҖдёӘжҠҪиұЎз»„д»¶пјҢиҮӘиә«дёҚдјҡжёІжҹ“дёҖдёӘDOMе…ғзҙ пјҢд№ҹдёҚдјҡеҮәзҺ°еңЁзҲ¶з»„д»¶й“ҫдёӯ

еҢәеҲ«пјҡ

  • v-ifеңЁеҲҮжҚўиҝҮзЁӢдёӯдјҡй”ҖжҜҒе№¶еҲӣе»ә
  • v-showеңЁйҰ–ж¬ЎеҠ иҪҪз»„д»¶ж—¶пјҢдјҡдёүдёӘеҗҢж—¶еҲӣе»ә
  • keep-aliveдјҡеңЁеңЁеҲҮжҚўж—¶еҖҷеҲӣе»әпјҢдҪҶжҳҜдёҚдјҡй”ҖжҜҒ

mixin

ж··е…Ҙ (mixin) жҸҗдҫӣдәҶдёҖз§ҚйқһеёёзҒөжҙ»зҡ„ж–№ејҸпјҢжқҘеҲҶеҸ‘ Vue з»„д»¶дёӯзҡ„еҸҜеӨҚз”ЁеҠҹиғҪгҖӮдёҖдёӘж··е…ҘеҜ№иұЎеҸҜд»ҘеҢ…еҗ«д»»ж„Ҹз»„д»¶йҖүйЎ№гҖӮеҪ“з»„д»¶дҪҝз”Ёж··е…ҘеҜ№иұЎж—¶пјҢжүҖжңүж··е…ҘеҜ№иұЎзҡ„йҖүйЎ№е°Ҷиў«вҖңж··еҗҲвҖқиҝӣе…ҘиҜҘз»„д»¶жң¬иә«зҡ„йҖүйЎ№гҖӮ

  • mixinеҜ№иұЎеҲҷlocalеҜ№иұЎиҰҶзӣ–mixinеҜ№иұЎ
  • з”ҹе‘Ҫе‘Ёжңҹй’©еӯҗеҮҪж•°еҲҷжҳҜеҗҲе№¶жү§иЎҢпјҢе…Ҳжү§иЎҢmixinеҗҺжү§иЎҢlocal
