# 前端错误的分类

一般来说前端错误分为两类

  1. 即时运行错误
  2. 资源加载错误

# 错误捕获方式

# 即时运行错误

  1. try catch
  2. window.onerror

# 资源加载错误

资源加载错误不会冒泡，所以 window.onerror 捕获不到

  1. object.onerror
  2. performance.getEntries ()（获取所有加载资源）
  3. Error 事件捕获

# 跨域 js 的运行错误

所有跨域 js 的运行错误都会显示为 Script error
解决方法：

  1. 在 script 标签上增加 crossorigin 属性 (客户端)
  2. 设置资源响应头 Access-Control-Allow-Origin:*（服务端）
