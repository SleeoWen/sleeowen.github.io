# д»Җд№ҲжҳҜ DNS йў„и§Јжһҗпјҹ

жҲ‘д»¬еңЁиҝӣиЎҢеүҚз«ҜжҖ§иғҪдјҳеҢ–ж—¶пјҢж¶үеҸҠеҲ° DNS ж–№йқўзҡ„дјҳеҢ–дёҖиҲ¬жңүдёӨзӮ№пјҡ дёҖдёӘжҳҜеҮҸе°‘ DNS зҡ„иҜ·жұӮж¬Ўж•°пјҢеҸҰдёҖдёӘе°ұжҳҜиҝӣиЎҢ DNS йў„и§Јжһҗ гҖӮ

DNS е®һзҺ°еҹҹеҗҚеҲ° IP зҡ„жҳ е°„гҖӮйҖҡиҝҮеҹҹеҗҚи®ҝй—®з«ҷзӮ№пјҢжҜҸж¬ЎиҜ·жұӮйғҪиҰҒеҒҡ DNS и§ЈжһҗгҖӮзӣ®еүҚжҜҸж¬Ў DNS и§ЈжһҗпјҢйҖҡеёёеңЁ 200ms д»ҘдёӢгҖӮй’ҲеҜ№ DNS и§ЈжһҗиҖ—ж—¶й—®йўҳпјҢдёҖдәӣжөҸи§ҲеҷЁйҖҡиҝҮ DNS Prefetch жқҘжҸҗй«ҳи®ҝй—®зҡ„жөҒз•…жҖ§гҖӮ

DNS Prefetch жҳҜдёҖз§Қ DNS йў„и§ЈжһҗжҠҖжңҜпјҢеҪ“жөҸи§ҲзҪ‘йЎөж—¶пјҢжөҸи§ҲеҷЁдјҡеңЁеҠ иҪҪзҪ‘йЎөж—¶еҜ№зҪ‘йЎөдёӯзҡ„еҹҹеҗҚиҝӣиЎҢи§Јжһҗзј“еӯҳпјҢиҝҷж ·еңЁеҚ•еҮ»еҪ“еүҚзҪ‘йЎөдёӯзҡ„иҝһжҺҘж—¶е°ұж— йңҖиҝӣиЎҢ DNS зҡ„и§ЈжһҗпјҢеҮҸе°‘з”ЁжҲ·зӯүеҫ…ж—¶й—ҙпјҢжҸҗй«ҳз”ЁжҲ·дҪ“йӘҢгҖӮ

# дёәд»Җд№ҲдҪҝз”Ё DNS йў„и§Јжһҗпјҹ

DNS дҪңдёәдә’иҒ”зҪ‘зҡ„еҹәзЎҖеҚҸи®®пјҢе…¶и§Јжһҗзҡ„йҖҹеәҰдјјд№ҺеҫҲе®№жҳ“иў«зҪ‘з«ҷдјҳеҢ–дәәе‘ҳеҝҪи§ҶгҖӮзҺ°еңЁеӨ§еӨҡж•°ж–°жөҸи§ҲеҷЁе·Із»Ҹй’ҲеҜ№ DNS и§ЈжһҗиҝӣиЎҢдәҶдјҳеҢ–пјҢе…ёеһӢзҡ„дёҖж¬Ў DNS и§ЈжһҗйңҖиҰҒиҖ—иҙ№ 20-120 жҜ«з§’пјҢеҮҸе°‘ DNS и§Јжһҗж—¶й—ҙе’Ңж¬Ўж•°жҳҜдёӘеҫҲеҘҪзҡ„дјҳеҢ–ж–№ејҸгҖӮDNS йў„и§Јжһҗ жҳҜи®©е…·жңүжӯӨеұһжҖ§зҡ„еҹҹеҗҚдёҚйңҖиҰҒз”ЁжҲ·зӮ№еҮ»й“ҫжҺҘе°ұеңЁеҗҺеҸ°и§ЈжһҗпјҢиҖҢеҹҹеҗҚи§Јжһҗе’ҢеҶ…е®№иҪҪе…ҘжҳҜдёІиЎҢзҡ„зҪ‘з»ңж“ҚдҪңпјҢжүҖд»ҘиҝҷдёӘж–№ејҸиғҪ еҮҸе°‘з”ЁжҲ·зҡ„зӯүеҫ…ж—¶й—ҙпјҢжҸҗеҚҮз”ЁжҲ·дҪ“йӘҢ гҖӮ
